Hello, AAU!

This time, a little different.

This page has Bulma added, a CSS framework that adds predefined styles.

Create new content or reuse your content from the index.html site. here inside the <div class="container"> element and design it using the styles and elements added by Bulma.

Try to use at least:

  • A Hero element
  • A Button element (without function)
  • A Card component
  • A Image figure element
  • A Notification element
  • Different colors using the Color Palette Helpers

You can still add your custom CSS stylings to style_with_framework.css.
Some standard HTML elements like <ul> may be needed to be put inside a <div class="content"> element to be styled correctly.
